Match Deletion

As an adminstrator you can delete matches. Only delete matches for erroneous submissions. Do not delete matches to get rid of team history as this will go against our commitment to record keeping and statistics. To delete matches, go to the submitted games' match page.

Deletions will revert the team to prematch status. The teams, players, and coaches will lose all stats gained from the match. DO NOT DELETE IF IT WILL UNSYNC CYANIDE TEAM AND OFTL TEAM.

"Undoing" Match Deletion

Resubmitting a match will "undo" a deletion, but the text file located in the coach's replay directory has to be edited. 1.) Have the coach watch the match replay for submission. 2.) Go into the replay directory and rename the .txt file (named something like oftl-new-submitted.txt) to anything else. 3.) Submit the match. 4.) A new text file will be created. Delete this and then rename the file you changed to it's original name. Done!
